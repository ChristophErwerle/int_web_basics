HTML Quiz Questions and Answers

  1. What does HTML stand for?
    Answer: A) HyperText Markup Language
  2. What is the purpose of the <head> tag in HTML?
    Answer: B) To contain metadata like title, character set, and link to stylesheets
  3. What is an HTML editor used for?
    Answer: C) To edit and write HTML code
  4. Which of the following is the correct HTML element for inserting a line break?
    Answer: A) <br>
  5. Which tag is used to create a hyperlink in HTML?
    Answer: B) <a>
  6. How do you create an unordered list in HTML?
    Answer: B) <ul>
  7. Which of the following is an HTML attribute?
    Answer: A) <href>
  8. What is the default alignment of the <h1> heading in HTML?
    Answer: B) Left
  9. Which tag is used to define a paragraph in HTML?
    Answer: A) <p>
  10. How do you add a comment in HTML?
    Answer: A) <!-- comment -->
  11. What is the correct HTML element to set a background color?
    Answer: B) <body bgcolor="color">
  12. Which HTML element is used for a quote or citation?
    Answer: D) <blockquote>
  13. Which tag is used to insert an image in HTML?
    Answer: A) <img>
  14. What is a favicon in HTML?
    Answer: A) A special image displayed in the browser's title bar
  15. Which tag is used to define a table in HTML?
    Answer: A) <table>
  16. How do you add a title to your HTML page?
    Answer: A) <title>
  17. What does the <th> tag represent in an HTML table?
    Answer: A) Table heading cell
  18. How do you define a table row in HTML?
    Answer: A) <tr>
  19. What does the colspan attribute do in an HTML table?
    Answer: B) Defines the number of columns a cell should span
  20. How can you add padding inside a table cell in HTML?
    Answer: C) style="padding:10px;"
  21. Which tag is used to create an ordered list in HTML?
    Answer: A) <ol>
  22. What is the purpose of the rowspan attribute in an HTML table?
    Answer: A) It defines the number of rows a cell should span
  23. Which HTML element is used for a definition list?
    Answer: A) <dl>
  24. What is the purpose of the colgroup element in an HTML table?
    Answer: B) To group columns for styling
  25. How do you make a table have borders in HTML?

  26. What tag is used to define a hyperlink target in HTML?
    Answer: B) target
  27. Which HTML tag is used to display a horizontal line?
    Answer: C) <hr>

    Answer: B) <select>

  31. Which attribute is used to specify the width of an image in HTML?
    Answer: C) width

    Answer: B) <ol>

    Answer: D) style="..." (within tags)
  34. Which tag is used to define a table cell in HTML?

  35. How do you define a form in HTML?

  36. Which tag is used to add a title to a webpage?
