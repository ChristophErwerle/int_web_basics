Unit Title Description
1 Introduction to Web Technologies
  • Understanding the concept of Internet and its different types of connection
  • Understanding of different browsers and websites
  • Creating email, file uploading and downloading
  • Surfing different social networking sites
  • Basics of HTTP and HTTPS
2 Introduction to HTML
  • Practical on HTML Structure
  • Practical on different basic tags and formatting tags
  • Practical based in link, list tag, images, audio-video tag and marquee tag
  • Create the table using TD, TR, TH, Colspan and Rowspan
  • Create inline frame, frame set, horizontal and vertical frameset.
  • Create HTML forms

  • working with Cascading Style Sheet : Syntex, measurement Units, colors, Background font and Text
  • Working with CSS – images, links , tables and borders

  • ID and class selector
  • ID and Class Selector, Inline,internal and External Stylesheet
4 Advanced CSS and CSS3
  • Working with CSS – Visibility, Positioning, layers, Pseudo Classes, Text effects, Media Type, Aural Media, Layouts and printing
  • CSS – Rounded corner , Border Image, Multibackground
  • CSS3 – color, gradients, shadows, text, 2D and 3D Transform
  • MCSS3 - Multicolumns, User Interface and Box sizing
  • Responsive Design with CSS Examples
5 Advanced Web Technologies
  • Definition of API and web service
  • Examples and uses
  • Data formats: JSON, XML
  • Implementation and security (API keys, tokens, OAuth)
  • Documentation examples
  • Web security: authentication, access control, encryption, input validation, patch management, WAFs
6 Client and Server-Side Scripting
  • Client-side scripting, form validation, data handling, AJAX and JSON
  • Using APIs
  • Front-end frameworks (React, Angular)
  • Server-side programming with PHP, Python, Ruby
  • Handling form submissions, user input, databases (MySQL, MongoDB)
  • Back-end frameworks (Express, Django)
  • Collaborative web development project implementing a real-world web application
7 Mobile Web Development
  • Introduction to mobile web development
  • Growth of mobile device usage and importance of responsive design
  • Differences between mobile web and native apps
  • Pros, cons, and use cases for each
  • When to prefer mobile web development
